# Project 2: Statistical Plots & Distribution Analysis

## Internship Project – Syntecxhub

### Objective
Analyze data distributions using statistical plots to identify skewness,
spread, and outliers.

### Tasks Performed
- Histograms and KDE plots for distribution analysis
- Boxplots for outlier detection
- Region-wise comparison of sales
- IQR-based outlier identification
- Exported visual insights

### Tools Used
- Python
- Pandas
- Matplotlib

### Key Insights
- Sales data shows right skewness
- Presence of significant outliers
- Regional differences in sales distribution
